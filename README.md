# shikaku.koplugin

A Shikaku puzzle plugin for [KOReader](https://github.com/koreader/koreader).

## Rules

Divide the grid into rectangles. Each rectangle contains exactly one number equal to its area (width × height in cells). Every cell must belong to exactly one rectangle.

## Features

- **Multiple grid sizes**
- **Three difficulty levels** — Easy, Medium, Hard
- **Rectangle preview** — see the rectangle as you draw it
- **Check** — highlights overlapping or incorrectly sized rectangles
- **Auto-save** — puzzle state saved and restored on next launch

## Installation

1. Download `shikaku.koplugin.zip` from the [latest release](../../releases/latest).
2. Extract into the `plugins/` folder of your KOReader data directory.
3. Restart KOReader.
4. Open the menu → **Tools** → **Shikaku**.

## Controls

| Action | How |
|--------|-----|
| Draw a rectangle | Tap start cell, then end cell |
| Remove a rectangle | Tap inside it |
| Check progress | Tap **Check** |
| New puzzle | Tap **New** |
| Show rules | Tap **Rules** |
